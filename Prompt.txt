Architecture Highlights
Canvas: char canvas[25][60] — _ background, * shapes, with a display_canvas() function that renders it live.

Drawing Algorithms:

Line → Bresenham's Line Algorithm
Rectangle → 4 × Bresenham lines (sides)
Circle → Midpoint Circle Algorithm
Triangle → 3 × Bresenham lines (vertices)
Object Model: Each shape stored as a Shape struct with an ID, type, visibility flag, and parameters. redraw_all() resets the canvas and re-rasterises every visible object after deletes/modifies.

Menus: Built with the Windows Console API (wincon.h) — coloured, cursor-positioned, interactive navigation using arrow keys or number shortcuts. No external libraries needed.
